Features




Drag Drop HTML5 App builder

A simple and intuitive interface lets you design your app and quickly iterate on changes. You can collaboratively edit the design with your team members.You can select from multiple preset templates to suit the requirements of your app. Also you can choose from a number of themes and UI styles.


Sharing and Testing

When your app is ready for testing, you can share the buildgap link with beta testers. All new changes will be available to testers as soon as you make them.


Phonegap Export / Native code

Buildgap makes it very easy to creative native apps from your projects. It lets you export source code which can be directly imported into XCode or Android Studio to create native apps.
